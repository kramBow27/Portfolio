PostSheet - Effortlessly send personalized emails & text messages

Clique aqui para ver o relato de Joe - CEO do PostSheet

PostSheet é uma aplicação web norte-americana feita no ambiente node.js com TypeScript, rendering com EJS e que viabiliza, de maneira simples, o envio de emails através de templates de dados. O aplicativo obtêm as informações a partir de tabelas (.CSV, .XLS, GoogleSheets, Airtable, .XLSX) trazendo um identificador a partir da primeira linha da tabela, e associando os dados das demais linhas as suas respectivas colunas.

Por fim, esses dados podem ser utilizados na construção de emails personalizados de acordo com os dados obtidos nas tabelas referentes a cada destinatário.

PostSheet é um projeto que foi sendo desenvolvido por uma sequência de desenvolvedores full-stack, mas nunca mais do que um, dessa forma, sou atualmente o quarto desenvolvedor a trabalhar no projeto porém o que trabalhou mais tempo.

A aplicação é utilizada por algumas instituições de renome como "The City University of New York" e "Stanford University" e ainda segue em desenvolvimento.

As últimas implementações realizadas por mim foram:

- Permitir .CSV e .XLS como data-sources para a criação das campanhas

- Integração com Stripe para direcionamento do usuário ao portal

- Alterações/Ajustes de front end

Também foram realizadas diversas correções de bugs que haviam sido deixados em versões anteriores. Devido a atual prioridade ao cumprimento de prazos para lançamento dos novos features, escolhi não alterar o stack, mas paralelamente as implementações principais está sendo realizada uma migração do front-end EJS para React, no que também se fará necessária a criação de um novo back-end, dado que EJS é uma tecnologia server-side-rendering e React client-side.

E mesmo que tivessemos adotado a abordagem de "React Server-Side Rendering" substanciais alterações teriam que ser feitas no back-end de qualquer maneira. Portanto, aproveitando a oportunidade, iremos reorganizar toda a estrutura do projeto para que uma futura implementação de features que envolvam funções complexas de data-analysis não se torne desgastante e complicada.
